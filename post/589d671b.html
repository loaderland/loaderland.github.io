软件

  • 系统：Armbian_23.08.0-trunk_Hisense-hs530r_jammy_edge_6.5.5_xfce_desktop.img**[GUI]**

  • 驱动：DriverAssitant_v5.12 只支持win10及以下

  • 刷机工具：海信可用Armbian_21.08.1_sm3399mb_focal_current_5.10.60

遇到的问题

谨慎更新

系统安装完成之后第一时间update + upgrade之后reboot发现无法进入系统，屏幕花屏，需要强制重新刷机

armbian系统不能随便更新 即apt-get update + upgrade

做的一些工作：初始化系统之后将现有的所有待更新包屏蔽自动更新，经过实测update+upgrade已经可以reboot二次进系统

# 装机完成之后查看可更新包
root@hisense-hs530r:~# apt list --upgradable
Listing... Done
armbian-config/jammy,jammy 23.8.3 all [upgradable from: 23.08.0-trunk]
armbian-firmware/jammy,jammy 23.8.3 all [upgradable from: 23.08.0-trunk]
armbian-jammy-desktop-xfce/jammy,jammy 23.8.1 all [upgradable from: 23.08.0-trunk]
armbian-plymouth-theme/jammy,jammy 23.8.3 all [upgradable from: 23.08.0-trunk]
armbian-zsh/jammy,jammy 23.8.3 all [upgradable from: 23.08.0-trunk]
base-files/jammy 23.8.3-jammy arm64 [upgradable from: 23.08.0-trunk-jammy]
fake-ubuntu-advantage-tools/jammy,jammy 23.8.3 all [upgradable from: 23.08.0-trunk]

# 禁止这些包的自动更新
sudo apt-mark hold armbian-config
sudo apt-mark hold armbian-firmware
sudo apt-mark hold armbian-jammy-desktop-xfce
sudo apt-mark hold armbian-plymouth-theme
sudo apt-mark hold armbian-zsh
sudo apt-mark hold base-files
sudo apt-mark hold fake-ubuntu-advantage-tools
sudo apt-mark hold linux-dtb-edge-rockchip64
sudo apt-mark hold linux-image-edge-rockchip64
sudo apt-mark hold linux-libc-dev

存储扩容

自带的存储大小为16G，想要通过插外置存储扩容root分区

在实际操作中发现并未使用lvm管理磁盘，即mmcblk0未建pv，而pv的创建需要格式化磁盘，因此无法通过lvm 逻辑卷手段扩容root分区，目前没有找到办法直接扩容，磁盘挂载等内容参考Linux LVM相关
